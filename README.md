# Azure VM + Network Deployment with Terraform

This repository contains Terraform scripts to deploy a basic Azure environment, including:

- Virtual Machine
- Virtual Network (VNet)
- Network Security Group (NSG)
- Storage Account

## Skills Demonstrated
- Infrastructure as Code (Terraform)
- Azure CLI / Azure Resource Management
- Networking & Security in Azure
- Hands-on cloud lab experience

## Usage
1. Clone this repo
2. Configure your Azure subscription credentials
3. Run `terraform init` → `terraform plan` → `terraform apply`
4. Observe the deployed resources in the Azure portal

This repository contains Terraform scripts to deploy a basic Azure environment, including a virtual machine, virtual network, network security group, and storage account. The project is intended as a hands-on lab for practicing cloud automation.
